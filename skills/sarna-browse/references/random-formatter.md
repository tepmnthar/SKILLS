# Random Mode Output Formatter

Use this template when presenting a random wiki page result in the conversation.

## Rules

- Links must be plain text, never markdown. Use `URL: https://...` on its own line.
- Preserve the original page's paragraph structure in the overview.
- Do not translate unless the user explicitly requests it.

## Template

```
# Sarna Random — {yyyy-MM-dd}

## {Page Title}
URL: {full_page_url}

{Overview paragraph 1}

{Overview paragraph 2}

{Overview paragraph 3}
...
```

## Example

```
# Sarna Random — 2026-05-21

## BattleTech Record Sheets: 3025 & 3026
URL: https://www.sarna.net/wiki/BattleTech_Record_Sheets:_3025_%26_3026

This reference manual catalogs gameplay tracking documents for military hardware deployed during the Succession Wars timeline. Bryan Nystul authored the work, which FASA released in 1996 with cover illustrations by Les Dorscheid. The publication spans 224 pages.

The volume supplies standardized and modified tracking cards for combat walkers alongside armored transport models primarily featured in companion technical manuals from the early 3020s. Developers utilized early drafting software called Mech Designer to generate these specific layout documents.

The publication organizes its tracking cards by operational classification, opening with aerial transport craft before detailing wheeled, tracked, and hovering ground vehicles. It then covers aquatic naval craft, followed by an extensive catalog of humanoid combat machines.
```
